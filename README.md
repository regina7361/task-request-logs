# Task-Request-Log

## Description
TRL is a task management application that allows user to create tasks to manage work project, schools projects or home projects.

## Live Application
**https://task-request-log.herokuapp.com/home**

## Technologies
    MySQL
    Heroku
    Handlebars
    Dotenv
    JawsDB
    Express
